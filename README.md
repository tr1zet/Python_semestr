## 📋 О проекте

Aura Weather — это современное веб-приложение для получения информации о погоде в любой точке мира. Проект реализует 4 различных механизма получения и обработки данных в соответствии с техническим заданием.

### 🎯 Функциональность

| № | Функция | Механизм работы |
|---|---------|-----------------|
| 1 | 🌡️ Текущая погода | HTTP-API (OpenWeatherMap) |
| 2 | 📅 Прогноз на 5 дней | HTTP-API (OpenWeatherMap) |
| 3 | 💭 Цитата о погоде | Скрапинг HTML (BeautifulSoup4) |
| 4 | 📋 Логирование действий | Файловая система ({user_id}.log) |
